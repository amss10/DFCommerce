## рҹ”„ Smart View Switching

### Scenario 1: User Lands on Homepage
```
1. Page loads
2. API fetches 62 products
3. JavaScript groups by category
4. Renders 6 carousels
5. User sees category browsing experience
6. Carousels show ~5 items each
7. Can scroll left/right with arrows
8. Can click "See All" for any category
```

### Scenario 2: User Searches
```
1. User enters search term
2. Hits Search button
3. Filter is applied to state
4. renderProducts() detects active filter
5. Switches from carousel to grid view
6. Grid shows only matching products
7. User can still filter by category
8. "Reset" button clears search
```

### Scenario 3: User Filters by Category
```
1. User selects category from dropdown
2. Clicks Apply button
3. Filter is applied to state
4. View switches to grid
5. Shows only products from that category
6. User can still search within category
7. "Reset" clears filters and shows carousels again
```
